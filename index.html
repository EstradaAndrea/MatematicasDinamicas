Matemáticas Dinámicas | Andrea Estrada Escánez

Bienvenido a Matemáticas Dinámicas, una web interactiva diseñada para explorar conceptos matemáticos a través del lenguaje HTML, CSS y JavaScript. Aquí aprenderás a construir contenidos matemáticos visuales y funcionales: desde la creación de listas y tablas básicas hasta la implementación de métodos numéricos, uso de lienzos interactivos y álgebra simbólica en el navegador. Cada ejercicio está enfocado a combinar la programación web con el pensamiento matemático, favoreciendo la creatividad, la comprensión y la práctica activa. ¡Explora, experimenta y diviértete programando matemáticas!

Contenidos

Ejercicio 2

Listas y Tablas HTML

Aprende a crear y personalizar listas ordenadas y desordenadas, así como tablas con estilos y elementos de entrada básicos.

Ejercicio 3

Estilos y MathJax

Aplica estilos CSS a tu página web, tanto en línea como desde un archivo externo, e integra MathJax para escribir expresiones matemáticas con LaTeX.

Ejercicio 4

Introducción a JavaScript

Comienza a programar con JavaScript modificando elementos HTML, trabajando con variables, condicionales, bucles, listas, funciones y depuración en consola.

Ejercicio 5

HTML + JavaScript + Canvas

Integra JavaScript con HTML para manipular texto, renderizar fórmulas matemáticas y dibujar en un lienzo utilizando eventos de ratón.

Ejercicio 6

Álgebra simbólica con JavaScript

Explora el uso de la biblioteca algebra.js para trabajar con fracciones, expresiones y ecuaciones algebraicas, incluyendo evaluación de expresiones y resolución simbólica.

Ejercicio 6.1

Demostración geométrica

Visualiza cómo la suma de fracciones infinitas rellena completamente un cuadrado de área 1.
